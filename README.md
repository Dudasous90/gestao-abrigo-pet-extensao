#  PetCare Manager - Projeto Extensionista (ODS 15)

Este sistema foi desenvolvido para a atividade extensionista da faculdade, focado na **Gestão de Abrigos de Animais de Rua**.

##  Relação com a ODS 15 (Vida Terrestre)
O abandono de animais domésticos impacta o equilíbrio da fauna silvestre e a saúde do solo. Este sistema auxilia no controle sanitário e populacional, protegendo o ecossistema local.

## Tecnologias Utilizadas
- Replic
- Express (API REST)
- Sequelize & SQLite (Banco de dados)

## Como o projeto ajuda a comunidade?
Oferece uma ferramenta gratuita para que protetores independentes organizem resgates, vacinas e facilitem o processo de adoção responsável.
